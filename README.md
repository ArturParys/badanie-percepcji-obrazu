# Badanie percepcji obrazu – Poltex 1969

Projekt badawczo-edukacyjny poświęcony analizie percepcji fotografii dokumentalnej
z wykorzystaniem map kliknięć (Microsoft Clarity).

## Cel
Celem projektu jest zbadanie, które elementy pojedynczej fotografii
najsilniej przyciągają uwagę odbiorców w warunkach mobilnych.

## Metoda
- jedna fotografia wyświetlana na pełnym ekranie
- instrukcja ograniczająca liczbę kliknięć
- analiza map kliknięć (heatmaps)
